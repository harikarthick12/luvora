# Snuggle Musix v5.26.3

- **Bug Fix**: Room ID is now properly visible and easily copyable when creating a Listen Together room.
- **Bug Fix**: Fixed Queue behaviour so that queued songs play as intended, and "Play Next" inserts correctly without scrambling the queue.
- **UI Enhancement**: Bottom navigation bar is now a sleek full-width rectangle instead of a pill.
- **UI Enhancement**: Search bar now auto-focuses instantly when switching to the Search tab, bringing up the keyboard automatically.
- **UI Enhancement**: Added several vibrant new Theme colors (Pink, Lavender, Mint, Peach, Midnight) to choose from in Settings > Appearance > Theme Colors.
- **Under the hood**: Enhanced handling for regional language playback. *Tip: For best regional recommendations (e.g. Tamil), ensure your Content Language is set correctly in Content Settings, or sign in!*
